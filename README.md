# Coffee-Shop-Analysis

## Project Overview
This project analyzes coffee shop sales data to optimize staffing efficiency and product focus. Inspired by prior barista experience, the report identifies key patterns in peak hours and product performance to provide actionable business recommendations.

## Tools & Skills Applied
* **Problem Definition:** Defined critical questions regarding staffing, product mix, and store performance.
* **SQL Skills:** Processed transaction data and extracted hour values for time-based analysis.
* **Tableau Visualization:** Designed interactive dashboards and heatmaps to reveal sales trends.
* **Analytical Thinking** Translated data findings into operational strategies suggestions and structured reporting.
